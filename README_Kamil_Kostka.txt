Kamil Kostka's Personal Website and Online Resume
Live Site: https://kamil-kostka-cv.web.app/
GitHub Repository: https://github.com/KaloeSan/cvWebsite_firebase/tree/master

Project Description
This project serves as a personal website and online resume for Kamil Kostka. It is designed to showcase my skills and experience as an IT professional, providing a platform for potential employers to learn more about my qualifications and contact me directly. The website is built using HTML, CSS, and JavaScript and is hosted on Firebase, ensuring reliable and fast access.

Key Features
Resume: A detailed resume section highlighting my work experience, education, and certifications.
Skills: A comprehensive list of both technical and soft skills.
Contact Form: An integrated form allowing visitors to get in touch with me directly through the website.
Mobile-friendly: The site is fully responsive, ensuring a seamless experience across different devices and screen sizes.
Dark Mode: A user-friendly feature that allows visitors to switch between light and dark themes based on their preference.

Technical Overview
Frontend Technologies:
HTML for the structure of the website.
CSS for styling, ensuring a visually appealing and responsive design.
JavaScript for interactivity and enhancing user experience, including the dark mode toggle.

Hosting:
Firebase provides robust hosting services, allowing for fast and reliable website performance.

Usage
Navigating the Website:
Use the navigation bar to access different sections of the website, such as Resume, Skills, Projects, and Blog.
Viewing the Resume:
Visit the Resume section to view detailed information about work experience, education, and certifications.
Exploring Skills:
Check out the Skills section for a comprehensive list of technical and soft skills.
Contacting Kamil:
Use the Contact Form to send a message directly to me.
Switching Themes:
Toggle between light and dark themes using the switch at the top right corner of the website.
